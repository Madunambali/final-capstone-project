HOW TO USE MOLECULAR EXPLORE

This guide will help you navigate the app and its features.

Home Page - Start exploring molecules with the "Start Exploring" button - Learn more about the app with the "Read About" link
Explore Page - Search for molecules by name or formula - Browse the molecular grid and hover for quick info - Click for detailed information
Predict Page - Start predicting molecular properties with the "Start Prediction" button - Enter molecular data or select from the library - View results and insights
Settings Page - Customize layout and theme settings - Save changes with the "Save Changes" button
Navigation Bar - Switch between Home, Explore, Predict, and Settings pages
Footer Links - Contact us for support or feedback - Check version info and privacy/terms
Tips - Explore thoroughly with search and filter features - Run predictions with different molecules and inputs - Stay updated with new features and settings Need help? Contact us at contact@molecularexplore.com.
